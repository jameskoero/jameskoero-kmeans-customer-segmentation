# K-Means Customer Segmentation

**Dataset:** Mall Customers (200 rows, Kaggle)
**Algorithm:** K-Means Clustering (K=5)
**Tools:** Python, scikit-learn, matplotlib, pandas
**Environment:** Google Colab (built on Android, Kisumu Kenya)

## Clusters Discovered

| Cluster | Name | Profile |
|---------|------|---------|
| 0 | Careful Spenders | High income, low spending |
| 1 | High Value Champions | High income, high spending |
| 2 | Average Segment | Mid income, mid spending |
| 3 | Enthusiastic Spenders | Low income, high spending |
| 4 | Budget Conscious | Low income, low spending |

## Key Business Insight

Cluster 0 — high earners barely spending — represents the single
biggest untapped revenue opportunity in this dataset.
A targeted loyalty program could unlock significant revenue.
